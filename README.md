# Oposiciones App

Aplicación estática de estudio para oposiciones de administración local, C2, C1 Administrativo y C1 Informática.

## Objetivo

- Estudiar por semanas.
- Crear reglas de memoria.
- Hacer test de blindaje.
- Guardar progreso con localStorage.
- Subir a GitHub Pages para estudiar desde el móvil.

## Estructura

- `index.html`: panel principal.
- `css/estilos.css`: estilos generales.
- `js/app.js`: progreso general.
- `datos/preguntas_constitucion.js`: preguntas tipo test.
- `temas/constitucion/articulo_01.html`: primer tema de estudio.

## Primera versión

Incluye Constitución Española, artículo 1.
